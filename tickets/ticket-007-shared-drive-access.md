# Ticket 007 Shared Drive Access Issue

## Ticket Summary

Ticket ID: 007  
Category: File Access  
Platform: Windows File Share  
Status: Resolved  

## User Reported Issue

The user reported that they could not access a shared drive used by their department.

## Investigation

Permissions were reviewed on the file server to confirm that the user account had the appropriate access rights.

## Click by Click Learning Process

1. Opened **File Explorer**.
2. Navigated to the shared drive location.
3. Checked folder permissions.
4. Opened **Security Properties**.
5. Verified user group membership.
6. Added the user to the required access group.

## Resolution

The user was added to the correct security group and access to the shared drive was restored.
